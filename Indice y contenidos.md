- 1. Hardware (La Base Física)
    
    Es todo lo tangible. El software no existe sin un soporte físico que ejecute la electricidad.
    
    - **Componentes:** Procesadores (CPU), tarjetas gráficas (GPU), memoria RAM y almacenamiento (SSD).
    - **Sistemas Embebidos:** Informática invisible en el mundo real (microchips en coches, electrodomésticos, IoT).
    - **Arquitectura de Computadores:** El diseño y optimización de estos componentes.
- 2. Infraestructura y Redes (La Conectividad)
    
    Es lo que permite que las máquinas hablen entre sí. El software actual no corre en el vacío, corre en red.
    
    - **Redes:** Cables submarinos, satélites, routers y protocolos (como TCP/IP) que mueven los datos por el mundo.
    - **Cloud Computing (La Nube):** Servidores gigantescos (AWS, Azure, Google Cloud) que alquilan potencia de cómputo y almacenamiento.
    - **Sistemas Operativos (El Puente):** Windows, Linux o macOS. Es el software base que traduce el hardware abstracto en algo que el software de usuario puede entender.
- 3. Software (La Lógica)
    
    Lo que tú ves, dividido en dos mundos principales:
    
    - **Desarrollo de Software:** * *Frontend:* Lo que el usuario toca (pantallas, apps, webs).
        - *Backend:* La lógica oculta, bases de datos y servidores que hacen que el frontend funcione.
    - **Sistemas de Base de Datos:** Motores que organizan, guardan y recuperan millones de datos de forma eficiente (SQL, NoSQL).
- 4. Datos e Inteligencia (El Activo)
    
    La informática moderna ya no solo ejecuta instrucciones programadas paso a paso, sino que procesa información para tomar decisiones.
    
    - **Ciencia de Datos / Big Data:** Analizar volúmenes masivos de información para encontrar patrones.
    - **Inteligencia Artificial y Machine Learning:** Algoritmos que aprenden de los datos en lugar de ser programados explícitamente.
- 5. Ciberseguridad (La Capa Transversal)
    
    No es una sección aislada, sino una capa que envuelve a todas las anteriores.
    
    - Protección de redes, encriptación de datos, defensa contra ataques y análisis de vulnerabilidades en el hardware y el software.
